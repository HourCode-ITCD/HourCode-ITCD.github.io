TYPE CHECKERS

A programming language feature that describes how variables are generated and how the language compiler or interpreter determines their types. A typechecker verifies that a construct's type matches the type required by its context. In Pascal, for example, the arithmetic operator mod requires integer operands, hence a type checker ensures that the operands of mod are of type integer. When generating code, type information acquired by a type checker may be required.

TYPESCRIPT

TypeScript is a programming language developed and maintained by Microsoft. It is a strict syntactical superset of JavaScript and adds optional static typing to the language. It is intended for the building of huge apps and transpiles to JavaScript.
