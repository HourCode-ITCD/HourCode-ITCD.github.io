Cascading Style Sheets (CSS)

Cascading Style Sheets (CSS) is a style sheet language used for specifying the display of a document written in a markup language such as HTML. CSS, together with HTML and JavaScript, is a cornerstone technology of the World Wide Web. CSS is intended to separate display from content, including layout, colors, and fonts. This separation can improve content accessibility; provide more flexibility and control in the specification of presentation characteristics; allow multiple web pages to share formatting by specifying the relevant CSS in a separate.css file, reducing complexity and repetition in the structural content; and allow the.css file to be cached to improve page load speed between the pages that share the file and its formatting.

BACKGROUND OF CSS

CSS is one of three foundational web technologies (the other two are HTML and JavaScript). CSS is an abbreviation for Cascading Style Sheets; the hints are in the terms 'cascading' and'style,' with cascading explaining how one style may cascade from one to another. One of the advantages of CSS is that several styles may be used inside the same HTML text. CSS is used to specify how HTML code will appear on a website. CSS (Cascading Style Sheets) is used to change the appearance of a web page, whereas HTML (Hypertext Markup Language) is used to provide content, including written text.
