GatsbyJS

Gatsby is a React-based open-source framework for creating websites and apps. It's great whether you're building a portfolio site or blog, or a high-traffic e-commerce store or company homepage.

  • Framework - A full-stack frontend framework that comes with everything you need out of the box.
  • Cloud - A unified cloud platform for building, previewing, and deploying your Gatsby projects.
  • Concierge - There are expert services that are tailored to help your team get the most out of your Gatsby deployment.

React is a library that is meant to provide a certain set of core functionality for developers to leverage. It is intended to be lightweight and broadly applicable. GatsbyJS, on the other hand, is a static progressive web app (PWA) generator that offers code and data splitting out of the box.

Gatsby allows the developers to make a site using React and work with any data source (CMSs, Markdown, etc) of their choice.

Getting Started with GatsbyJS

To get started using GatsbyJS, download the Gatsby CLI: npm i -g gatsby. This command-line tool enables you to generate, run, and build a Gatsby application. To get up and running quickly, you can use the Gatsby default starter repository. After cloning this, run npm install and then gatsby develop. This will generate a fully active Gatsby application running on http://localhost:3000.
