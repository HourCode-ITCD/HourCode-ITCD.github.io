Background of CSS

Why do we need CSS?

To begin with, CSS guarantees that your web pages are uniform. Consider a website with hundreds of pages. Now consider having to input the code to set header sizes, layout, and other display data, and then mixing it all in with the text each time you wished to create a new page. Also, consider having a website with hundreds of pages and being able to update only one of them while leaving the others the same — CSS allows you to do so. CSS provides uniformity where it is needed while also allowing you to make modifications to particular pages or parts.

Who invented CSS?

According to Wikipedia, the origins of CSS may be traced back to Norwegian Hkon Wium Lie, who attempted to build a universally uniform style sheet for the World Wide Web in 1994.

The Arena web browser was the first site on which Lie tested CSS. Lie went on to co-create CSS1, CSS2, and RFC 2318 versions with Tim Berners-Lee and Robert Cailliau. CSS, in all of its standards, became an established online standard in its first decade of existence (1994–2004), profoundly affecting the look and accessibility of the world wide web as we know it today. CSS3 was released in 1999.

Web standards are a subject dear to Lie's heart. He has petitioned key technological firms such as Microsoft and other browsers to embrace common web standards since providing CSS, and he continues to improve CSS's usage for online printing and pagination on displays.

What is the difference between CSS1, CSS2 and CSS3?

CSS (1996) allows the user to select font style and size and change the colour of the text and background.

CSS2 (1998) has capabilities that allows the user to design page layout.

CSS3 (1999) allows the user to create presentations from documents and to select from a wider range of fonts including those from Google and Typecast. Uniquely, CSS3 allows the user to incorporate rounded borders and use multiple columns.

How do we learn CSS?

CSS, as much as it is a useful tool for web developers, is also a language that anybody can learn, much like studying French or Japanese. Rather of memorizing every single line of code, a CSS reference site may be used to look up properties.
