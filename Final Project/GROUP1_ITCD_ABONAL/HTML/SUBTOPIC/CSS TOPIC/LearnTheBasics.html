Learn the Basics

Why learning CSS the right way is important

CSS becomes more of a trial and error process if you don't have a solid knowledge of the fundamentals. You'll experiment with different numbers for various attributes and eventually settle on one that comes near to what you're looking for without really knowing how it works.

You'll start searching simple things like "how to center two divs" or "how to align a div and a text vertically" and copy and paste code from StackOverflow or codePen every time. Back when flexbox wasn't that popular, the question was, "How do you position a div both vertically and horizontally in a page?" " was a well-known CSS interview question. Many beginners got the horizontal section correct, but just a handful got the vertical part correct as well.

The Basics of CSS

If you're new to web development, start with HTML basics before moving on to CSS. Read the theory on what CSS is, how it works in the browser, and its basic syntax and usage before diving into CSS.

Learn about the many types of stylesheets available, their distinctions, selectors, and fundamental styling such as font-size, width, and height, among other things.

CSS Box Model

Understand the fundamentals of the CSS box model and the attributes that go with it, such as margin, border, padding, and so on.

Images and Background

Images bring the website to life. There are several ways to include an image, including image tags, background colors/gradients, and background images to various other tags. You might also use what you've learned earlier, such as how to establish borders on photographs or how to use many images to create a basic gallery.

Display and Position

These are two of the most important attributes in CSS, and you must pay close attention to them in order to comprehend them correctly. Knowing these two characteristics intimately may make your CSS journey much easier.

Even skilled programmers can make errors in this property. Learn how each one works, how the position of one element impacts its siblings or parent, when you utilize them, and so on.

Float layouts are a bit out of date lately, but there are many old websites that still utilize them.

Colours, Fonts, lists and Tables

Understand different colour formats such as HEX code, rgb, rgba, hsl, hsla, transparent etc

Learn how to utilize various typefaces. Some fonts aren't supported by all browsers, therefore you'll have to learn how to manually install fonts with woff or ttf files or import Google Fonts.

Pseudo-classes and Combinators

A CSS pseudo-class is a keyword added to a selector that specifies a special state of the selected element(s). A pseudo-class can be as simple as :hover or :visited or something complex like :nth-last-of-type(odd)

Combinators allow us to quickly apply styles to child components or siblings without having to build new classes for each of them.

Debugging and Dev tools

CSS does not generate any errors. If you don't get the styles right, it quietly ruins the UI? As a result, understanding how to use Dev tools is critical.

Chrome Dev Tools is an excellent resource for web developers. It includes a slew of useful capabilities for debugging your website in real time, as well as performance monitoring tools like Lighthouse.

Practice

The contents you learnt above are enough to develop a basic website, so at this point you should start practicing by developing small websites.When developing a genuine application, you will face a variety of obstacles. You may design a small website for a resort, a picture gallery, a blog, or a few basic elements of your favorite social networking platform, such as Facebook or Instagram, for practice.
