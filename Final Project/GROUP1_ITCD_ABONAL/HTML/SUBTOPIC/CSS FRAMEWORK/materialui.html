Material UI

Material-UI is an open-source framework with React components that follow Google's Material Design guidelines. In a nutshell, is an open-source project that includes React components that apply Google's Material Design. It began in 2014, not long after React was released to the public, and has since risen in popularity. Material-UI is one of the most popular React user interface frameworks, with over 35,000 stars on GitHub.

Material UI provides a complete set of UI capabilities to assist you in releasing new features more quickly. Start with Material UI, our fully-loaded component library, or use our production-ready components with your own design system.

Usage

MUI components operate independently. They are self-sufficient and will only inject styles that they require. They make no use of global stylesheets like normalize css.

Any of the components shown in the documentation can be used. Please visit the demo page for each component to see how it should be imported.

Text buttons are typically used for less-pronounced actions, including those located: in dialogs, in cards. In cards, text buttons help maintain an emphasis on card content.

Contained buttons are high-emphasis, distinguished by their use of elevation and fill. They contain actions that are primary to your app.

Outlined buttons are medium-emphasis buttons. They contain actions that are important but aren't the primary action in an app.


Sizes and colors

The button group's appearance may be controlled using the size and color properties.

Sizes varies from small, medium and large with the use of the keyword size="*"

As for the colors, color="*" is used where keyword named colors can be used such as primary, secondary, success, error etc.
