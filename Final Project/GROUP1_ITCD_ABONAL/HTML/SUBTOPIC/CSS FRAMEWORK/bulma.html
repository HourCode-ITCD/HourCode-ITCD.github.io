Bulma

Bulma is a free, open source framework that comes with ready-to-use frontend components that can be combined to create responsive web interfaces. Bulma, which was first launched in 2016, has acquired enough support among front-end developers to compete with well-known CSS frameworks such as Bootstrap.

Bulma is based on Flexbox, a CSS layout concept that adapts the width of a page element dependent on its container's width. To design elements that comply to the Flexbox concept, developers must code both HTML and CSS. Flexbox behavior and the majority of other things can be managed with HTML classes in Bulma.

Bulma Setup

Get the bulma Cascading Style Sheet(CSS) File

To use Bulma, you can use the pre-compiled .css file by importing the css file from jsDelivr:

"@import "https://cdn.jsdelivr.net/npm/bulma@0.9.4/css/bulma.min.css";"

or the Right to left(RTL) directory version:

"@import "https://cdn.jsdelivr.net/npm/bulma@0.9.4/css/bulma-rtl.min.css";"


Get the Bulma Syntactically awesome style sheet(Sass) File

Another option is to install the Node Package Manager (NPM) package for bulma:

"npm install bulma"


Modifier Syntax

Most Bulma elements have alternative styles. To apply them, you only need to append one of the modifier classes. They all start with "is-" or "has-".

Buttons

Buttons can have 6 main colors with the use of "is-*" color such as:

"is-primary" - for blue green

"is-link" - for blue

"is-info" - for light-blue

"is-success" - for green

"is-warning" - for yellow

"is-danger" - for red


Size

It is also possible to alter the size of the button with the use of is-* size such as:

"is-small" - for small

"is-medium" - for medium

"is-large" - for large


Style or State

buttons can be displayed as outlined, loading or disabled with the use of the codes:

"is-outlined" - for outlined

"is-loading" - for loading

"is-disabled" - to disable

Responsiveness

Every piece of Bulma is designed with mobile in mind and is optimized for vertical reading, therefore on mobile:

  • Vertically stacked columns
  • The children of the level component will be piled vertically.
  • The navigation menu will be disabled.

The "is-mobile" modifier may be used to compel the horizontal layout for both columns and levels.

Breakpoints

Bulma includes four breakpoints that define five different screen sizes:

  • mobile: up to 768px
  • tablet: from 769px
  • desktop: from 1024px
  • widescreen: from 1216px
  • fullhd: from 1408px

Variables

Name Type Value
$gap size 32px
$tablet size 769px
$desktop computed 960px + (2* $gap)
$widescreen computed 1152px + (2 * $gap)
$fullhd computed 1344px + (2 * $gap)

Colors

Most elements and components have color variants courtesy to modifiers like is-primary and is-dark, which have the syntax is-$color. This is courtesy to the $colors Sass map, which Bulma uses to cycle through all of the colors and inverts.

Color Variable Value
White $white $white
Black $black $black
Light $light $white-ter
Dark $dark $grey-darker
Primary $primary $turquoise
Link $link $blue
Info $info $cyan

Warning $warning $yellow
Danger $danger $red

Bulma also includes a $hues Sass map with just greyscale shades between black and white.
