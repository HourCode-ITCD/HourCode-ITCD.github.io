Bootstrap

Bootstrap is a comprehensive toolkit that includes HTML, CSS, and JavaScript tools for creating and developing web pages and apps. It's a free and open source project that was built by and for Twitter and is hosted on GitHub. You've come to the correct spot if you're still not sure what Bootstrap is or want a deeper understanding of what it is and what it does well.

With Bootstrap, the world's most popular front-end open source toolkit, you can quickly develop and configure responsive mobile-first sites with Sass variables and mixins, a flexible grid system, vast prebuilt components, and powerful JavaScript plugins.

Buttons

The .btn classes are intended for usage with the <button> element. These classes can also be applied to <a> and <input> components, though some browsers may apply a slightly different rendering.

Instead of referring to new pages or sections inside the current page, button classes on <a> components that are used to initiate in-page functionality such collapsing content should be given a role="button" to properly express their purpose to assistive technology like screen readers.

Texts Transform

Transform text in components with text capitalization classes.

It's worth noting that .text-capitalize only affects the first letter of each word, keeping the case of the remaining letters unchanged.

Font Size

Change the font size of text quickly. Our heading classes (.h1 - .h6) use font-size, font-weight, and line-height, but these utilities just use font-size. These utilities are sized to match HTML's heading elements, so as the number of them grows, their size shrinks.

Font weight and italics

These programs allow you to quickly modify the font weight or style of text. Font-style utilities end in.fst-*, whereas font-weight utilities end in .fw-*.

Scaffolding

Certain HTML elements and CSS features in Bootstrap require the usage of the HTML5 doctype. Include it at the start of every project you work on.
