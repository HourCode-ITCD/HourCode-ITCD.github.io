Chakra UI

Chakra UI is a component library that provides you with the building blocks you need to create React apps. It's basic, modular, and easy to use. It's quite straightforward to use, especially if you're already familiar with ReactJs components.

Chakra UI is an accessibility-aware React components library. It has a contemporary design system that is simply expandable and adjustable. The easy styling API cuts development time in half, allowing developers to quickly prototype concepts and achieve the required style and "brand" by modifying only one file.

Installation

ChakraUI can be installed using Yarn or NPM in your respective directory. Inside the CRA project directory, install Chakra UI by running either of the following:

"yarn add @chakra-ui/react @emotion/react@^11 @emotion/styled@^11 framer-motion@^6"

"npm i @chakra-ui/react @emotion/react@^11 @emotion/styled@^11 framer-motion@^6"

It's a must to first include <ChakraProvider> in your index.js file for ChakraUI to be initialized.

  • fontSizes
